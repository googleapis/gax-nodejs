Class: EventEmitter

EventEmitter

new EventEmitter(callbacknullable)

EventEmitter is a custom version of event emitter for the API call.

In addition to normal events, it has a method 'cancel' to stop an ongoing event.

Parameters:
Name Type Attributes Description
callback APICallback <nullable>

the callback to be called when the API call ends.
Properties:

A promise resolves to the result value of the API (or fails when API call fails or is canceled).

Class: BundleEventEmitter

BundleEventEmitter

new BundleEventEmitter(executor, callback)

BundleEventEmitter manages the scheduling of bundled request.

This is an event emitter. It emits 'data' event on the data, and 'error' event when an error happens or the event emitter is canceled.

Parameters:
Name Type Description
executor BundleExecutor

The bundle executor for this event.
callback function

The callback to be called when this event has been fulfilled.
Properties:
Name Type Description
result Promise

The promise to be resolved to the result of the API call. This would be useful for waiting the result.

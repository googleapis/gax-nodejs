GrpcClient

new GrpcClient(optionsopt)

A class which keeps the context of gRPC and auth for the gRPC.

Parameters:
Name Type Attributes Description
options Object <optional>

The optional parameters. It will be directly passed to google-auto-auth library, so parameters like keyFile or credentials will be valid.

Properties
Name Type Attributes Description
auth Object <optional>

An instance of google-auto-auth. When specified, this auth instance will be used instead of creating a new one.
grpc Object <optional>

When specified, this will be used for the 'grpc' module in this context. By default, it will load the grpc module in the standard way.
